# TaskMasterAI

TaskMasterAIは、生成AIを利用してタスクの目標設定を行い、ExcelやWordに出力できるJavaベースのタスク管理アプリケーションです。

## 機能

- タスクの追加、削除、編集
- ExcelやWordへのエクスポート・インポート機能
- 生成AIとの連携によるタスク目標設定
- ポモドーロ・テクニックのサポート

## 始め方

### 前提条件

- Java 8以上
- MySQL
- Gradle
